# Perform Addition of Two Numbers in Java Without Using Scanner

## My Solution

```java
public class AddTwoNumbers {
    public static void main(String[] args) {
        // Define two numbers
        int num1 = 15;
        int num2 = 20;

        // Perform addition
        int sum = num1 + num2;

        // Print the result
        System.out.println("The sum of " + num1 + " and " + num2 + " is: " + sum);
    }
}
```

---

## Output

```plaintext
The sum of 15 and 20 is: 35
```
